# IronSight MR: Adaptive Spatial Intelligence for High-Pressure Training

**Concept & Prototype by:** Naga Avinash  
**Tech Stack:** Unity XR Foundation, ARCore, ARKit  
**Status:** In-Development / Concept Phase

---

## 🎯 The Vision
Current training systems rely on 2D screens (Manuals, Simulators, Dashboards). In high-pressure environments—like emergency response or industrial repair—there is a "Translation Gap" between 2D information and 3D action.

**IronSight MR** is a conceptual framework for an adaptive Mixed Reality system that embeds intelligence directly into the physical environment.

## 🧠 The Problem: The Translation Gap
Failure in high-stakes scenarios rarely stems from a lack of knowledge. It stems from:
* **Attention Drift:** Focusing on the wrong stimulus.
* **Cognitive Overload:** Data saturation during high heart-rate events.
* **Spatial Friction:** The mental lag of translating a flat map into a 3D environment.

## 🛠 Key Design Pillars
### 1. Attention Training
Moving beyond simple task-completion to simulate environmental noise (visual clutter, audio distractions), training the user to find the "signal."

### 2. Spatial Guidance (World-Locked UI)
Replacing floating menus with world-locked markers:
* **Dynamic Pathfinding:** Safety zones projected on the floor.
* **Object-Tethered Instructions:** Manuals digitally "stuck" to the physical hardware.

### 3. Adaptive Cognitive Loading (Logic Flow)
The system monitors user state (Gaze erraticness/Heart rate) to trigger **Focus Mode**. Non-essential UI elements fade out, leaving only the "Next Best Action."
